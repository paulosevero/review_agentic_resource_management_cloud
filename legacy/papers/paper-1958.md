### Abstract

Recently, multimodal temporal fusion has become a powerful method for achieving real-time video action recognition in smart surveillance systems to enhance the situational awareness and security. There are multiple data streams considered for recognition accuracy and robustness, which include RGB, optical flow, and audio, in this paper, we propose an Adaptive Multimodal Temporal Fusion Network (AMTF-Net) to combine these data streams. Using scene context, CMTF-Transformer dynamically chooses optimal fusion weights to perform cross modality temporal fusion that aligns features optimally in both space and time. Furthermore, we utilize a Graph Convolutional Network (GCN) to improve the human-object interaction modeling, and the Edge-AI optimization strategy acts to facilitate the real-time performance by a lightweight model pruning and hybrid cloud-edge processing. Finally, it includes a self-supervised learning adaptation to unseen action categories allowing for a reduction in dependence on labelled data. An Event Triggered Anomaly Recognition Module (ETARM) is integrated to identify dynamically suspicious behaviors that can improve anomaly detection. Experimental results verify that AMTF-Net achieves higher accuracy and efficiency than SOTA and consumes less latency for real-time deployments. To address crucial problems in surveillance, such as occlusions, environmental variations, and computational resource issues, the proposed approach is very suitable for smart surveillance applications in security monitoring, crowd control, and automated threat detection. It will also continue to extend the framework to handle the complementary multi-agent interactions and adversarial scenarios. © 2025 IEEE.
