### Abstract

Cloud Computing is a new model for delivering resources such as computing and storage on demand through a network. It has many advantages like reducing the cost, however, it remains the problem of security, which is the major brake for the migration towards Cloud Computing. Cloud Data Storage (CDS) is an important service of Cloud, it provides the way to store and manage data remotely in the Cloud, but data security remains the problem, which makes users worried about the confidentiality and the integrity of their data. In this paper, we propose a framework of security to ensure the CDS, which is based on agents. It contains two layers: Cloud Provider layer and the User layer. © Springer International Publishing AG, part of Springer Nature 2018.
