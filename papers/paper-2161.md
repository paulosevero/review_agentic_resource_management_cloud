### Abstract

Large language models (LLMs) have gained significant attention for their powerful capabilities and broad applicability. However, cloud-based deployment raises concerns over data privacy, security, and latency, limiting user experience. Edge computing offers a promising alternative by processing data closer to the source, reducing latency and enhancing privacy. Nonetheless, the constrained computational resource makes it difficult to handle the demands of large models independently. To address these challenges, this work proposes a deep reinforcement learning (DRL) framework that optimizes the execution of LLM applications in edge computing. The framework jointly considers task processing order, priority, offloading decisions, and resource allocation to minimize application duration. Leveraging a proximal policy optimization (PPO) algorithm with a graph convolutional network (GCN) module for feature extraction, the approach achieves efficient task scheduling. Experiments confirm its superior performance and convergence. © 2025 IEEE.
