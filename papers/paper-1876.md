### Abstract

This edge computing ear disease diagnosis system based on deep learning and a large language model achieves multimodal diagnosis using otoscope images and patient symptom text by integrating an efficient deep learning model and a large language model deployed on edge devices. The system architecture adopts a layered design and incorporates a dynamic resource allocation mechanism to optimize computational load and privacy protection. The core algorithm extracts image features using a convolutional neural network and incorporates text information using an attention mechanism to generate highly interpretable diagnostic reports. Experimental results demonstrate that the system excels in accuracy, real-Time performance, and low power consumption, achieving an overall accuracy of 94%, an inference time of 1.2 seconds, and power consumption as low as 0.45W, significantly outperforming baseline models. Performance analysis, through visualization, demonstrates the system's high discrimination across multiple ear disease classifications, providing reliable technical support for portable medical diagnosis and suitable for both clinical and primary care settings. © 2025 COPYRIGHT SPIE.
