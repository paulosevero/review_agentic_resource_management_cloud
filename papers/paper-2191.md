### Abstract

Fog-based Service Function Chaining (SFC) enables modular, low-latency network services but introduces new attack surfaces on distributed edge nodes. We present a Fog node testbed to evaluate four attacks-SSH brute-force, API manipulation, routing loop injection, and low-noise denial-of-service-each producing distinct system-level telemetry fingerprints. Using system metrics without packet inspection, we apply large language models (LLMs) for anomaly detection. Our LLaMA 3.1 8B model outperforms comparable opensource LLMs, achieving up to 93% detection accuracy, including against stealthy attacks. This approach offers a robust, scalable method to secure fog-based SFCs.  © 2025 IEEE.
