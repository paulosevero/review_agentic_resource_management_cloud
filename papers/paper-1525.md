### Abstract

The rapid evolution of cloud computing has revolutionized the IT landscape, provided unprecedented scalability and flexibility while introduced complex challenges in resource management, cost optimization, and performance tuning. The Dynamic Load Balancing in Cloud Infrastructure addresses these challenges with an innovative AI-driven solution for cloud resource allocation, load balancing, and cost management. Motivated by the need for intelligent, automated tools to manage modern cloud environments, this research introduces the dynamic load balancing in cloud Infrastructure e, leverages advanced artificial intelligence and machine learning techniques, including deep reinforcement learning to create a system capable of real-time adaptation to dynamic workloads and resource demands. The proposed methodology integrates real-time metrics analysis, predictive modelling, and adaptive decision-making algorithms, enabling informed decisions about resource scaling, load distribution, and cost optimization. Unlike prior studies that focus on isolated aspects of cloud management, our unified platform simultaneously addresses multiple optimization facets. A key innovation is the multi-agent reinforcement learning framework, which enables coordinated decision-making across optimization domains, balancing conflicting objectives such as minimizing costs and maintaining high performance. Additionally, our approach addresses high-dimensional state and action spaces, overcoming limitations of traditional reinforcement learning methods. Empirical evaluations demonstrate significant improvements, including a 27% reduction in cloud spending, a 35% improvement in application performance, and a 42% decrease in resource allocation conflicts across diverse workloads. The system's modular and extensible design also ensures compatibility with diverse cloud platforms and evolving infrastructure needs. These results validate the effectiveness of our AI-driven approach and highlight its potential to transform cloud management. Our scalable, adaptive solution provides a foundation for more efficient and autonomous cloud systems, addressing the complexities of modern cloud computing. infrastructure needs.  © 2025 IEEE.
