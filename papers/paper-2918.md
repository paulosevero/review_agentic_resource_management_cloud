**Abstract**

> This study examines user behavior evolution in the 5G short video ecosystem using a V‑SIbR dynamic model based on evolutionary game theory. It integrates system dynamics to nonlinearly couple network parameters like latency and bandwidth with behavioral variables such as interaction frequency and strategy selection. Experiments show that user interaction grows exponentially when end‑to‑end delay is below 20 ms, with a strategy switching rate 2.3 times that of linear models. The paper proposes a collaborative optimization mechanism for network slicing and behavioral strategies, achieving Pareto optimality in bandwidth allocation and user QoE through a multi‑agent game architecture. Compared with static models, this dynamic framework increases peak user interaction capacity by 43.2% and accelerates equilibrium convergence by 37.5% under high concurrency. It validates cross‑layer synergy and advances information ecosystem research from static analysis to dynamic evolutionary modeling, supporting intelligent decision‑making for 5G short video systems. © 2026 Taylor & Francis Group, LLC.
