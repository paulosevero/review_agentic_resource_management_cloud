### Abstract

Cloud platform has a strong computing power and storage resources, effective management of these resources has become a problem to be solved in the field of cloud computing. In order to effectively achieve the dynamic changeable autonomous and heterogeneous cloud resource management, this paper presents a multi-agent system of cloud computing resource management model, make full use of the advantages of multi-agent system, efficient cloud computing resource management. © 2016 IEEE.
