### Abstract

As user application demands continue to grow, edge computing has shown significant advantages in reducing latency and alleviating central computing burdens. To efficiently process computation-intensive tasks, an increasing number of end devices are choosing to offload tasks to edge servers. However, most existing studies assume that tasks are independent, overlooking the common task dependencies observed in real-world applications. This results in suboptimal performance for multi-step task execution. While some studies have introduced Directed Acyclic Graphs (DAGs) to model task dependencies, they primarily focus on single-user or single-agent scenarios, lacking a comprehensive framework for coordinating multi-source task topologies and cooperative scheduling. Unlike previous works, this paper targets a more realistic multi-agent offloading setting and proposes a joint-aware DAG scheduling mechanism to coordinate offloading decisions and resource allocation across multiple vehicular agents. © 2025 SPIE.
