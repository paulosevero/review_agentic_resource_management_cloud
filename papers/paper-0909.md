**Abstract**

> With the rise of large language models (LLMs), network management will require significantly less human intervention. This demo shows service deployment from a user request using simple natural language. We leverage LLM to decompose and translate the user request into TM Forum Intent representation. This intent is a blueprint for generating Kubernetes deployment manifest files, ultimately deploying the service to the designated location. This approach significantly reduces manual configuration and streamlines service deployment.  © 2024 IEEE.
