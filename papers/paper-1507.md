**Abstract**

> The growing demand for real-time communication (RTC) services necessitates more efficient and reliable infrastructure provisioning methods. This work-inprogress paper introduces Telcoformrix, an AI-native Infrastructure-as-Code (IaC) framework designed to simplify the deployment of RTC infrastructure. By leveraging a fine-tuned large language model (LLM), Telcoformrix translates high-level YAML declarations into Terraform JSON configurations, abstracting the complexities of cloud and edge network setup. Integrated with GitOps workflows, the framework aims to automate validation and deployment via CI/CD pipelines. This paper outlines the motivation, design, and planned evaluation of Telcoformrix, positioning it as a promising approach to reducing cognitive load and configuration errors in RTC infrastructure management. © 2025 IEEE.
