### Abstract

Efficient task scheduling in cloud systems is difficult because of different resources, changing workloads, and complex task meanings. Many reinforcement learning and graph-based methods cannot fully understand task semantics or handle time information well, which leads to poor scheduling. This paper presents HATAN-LLM, a simple and clear hierarchical attention task assignment framework. It combines semantic learning from large language models, relational reasoning from heterogeneous graph neural networks, and decision control from hierarchical reinforcement learning. The framework uses multiple LLM embeddings to learn how tasks and experts relate, builds a dynamic graph to model time and relation structures, and applies a policy network trained by constraint-aware gradients to keep decisions stable and correct. Meta-learning helps the model adjust when workloads change. HATAN-LLM improves SLA handling, load balance, and scalability across cloud systems. ©2025 IEEE.
