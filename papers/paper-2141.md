**Abstract**

> To address the issue of high latency in wireless transmission task offloading in mobile edge computing networks for Internet of Things (IoT) scenarios, this paper proposes a Multi-Agent Reinforcement Learning (MARL) algorithm. In this algorithm, each server is regarded as an agent, and each agent builds a dual-network architecture. It utilizes deterministic policies to generate action sequences and optimizes the loss function by combining the differences between action value functions and state value functions. By optimizing the sample sampling mechanism through an experience replay pool, the training convergence speed of individual agents is effectively enhanced. Simulation experiments show that compared with heuristic algorithms, MARL reduces latency by an average of 6.10 %; compared with nearest server allocation, it reduces latency by an average of 22.70 %; and compared with random server allocation, it reduces latency by an average of 24.27 %, while demonstrating stronger robustness. © 2025 IEEE.
