### Abstract

The Mobile Edge Computing (MEC) is proposed as an emerging paradigm for reducing the service latency and saving transmission bandwidth. Service-based application development greatly increases extensibility and maintainability. Compared with the cloud center, the MEC servers have limited computing and storage capabilities, which increases the difficulty of service deployment in the edge network. Since the MEC is a typical distributed system, each server independently updates the service deployment according to geographic differences. In order to improve the system utility, the cooperation between servers is indispensable. In this paper, we combine the multi-Agent Deep Reinforcement Learning (DRL) with auction mechanism, aiming to solve the multi-service placement problem in MEC systems, where each server makes the service bid and update decision based on the DRL, and the auction mechanism is used to coordinate the service placement between servers. The simulation results show that the introduction of collaboration and auction in proposed method can improve the system revenue.  © 2023 IEEE.
